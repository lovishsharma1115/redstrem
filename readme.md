# 🌟 Redstream — A Clean and Simple E-Commerce UI

Redstream is a small passion project I built to sharpen my frontend fundamentals.  
It’s a lightweight e-commerce UI — simple, clean, and built from scratch without relying on heavy frameworks.

No distractions. No backend.  
Just pure HTML, CSS, and JavaScript working together to create a smooth, modern shopping interface.

---

## ✨ Why I Built This

I wanted a project that feels practical, something that represents what a real website looks like—but still simple enough to experiment with.  

E-commerce is the perfect playground for learning:

- Layout structure  
- Responsive design  
- UI/UX basics  
- Clean CSS architecture  
- Reusable components  
- Product grid patterns  

Redstream helped me bring all these ideas together in one mini project.

---

## 🎯 Features

- Responsive homepage layout  
- Clean product card UI  
- Smooth grid that adapts to all devices  
- Simple JavaScript add-to-cart alerts  
- A neat navigation bar  
- Beginner-friendly structure  
- Easy to customize and expand  

---

## 🧰 Tech Stack

This project uses only the essentials:

- **HTML5** – for the page structure  
- **CSS3** – for design and responsiveness  
- **JavaScript (ES6)** – for basic interactivity  

No frameworks. Pure fundamentals.
